# 🦷 AI Dental Health Survey Guide

An interactive web application designed to assist participants in the **National AI Dental Health Survey** by providing clear instructions for capturing high-quality dental images while promoting awareness of oral hygiene and best practices.

Built using **React**, **TypeScript**, **Vite**, and **Tailwind CSS**, the application offers an engaging, responsive, and user-friendly experience.

---

## ✨ Features

- 🦷 Interactive dental survey guide
- 📸 Step-by-step dental image capture instructions
- 📖 Before-you-start preparation checklist
- ✅ Image capture best practices
- ⚠️ Common mistakes to avoid
- 💡 Oral health awareness section
- 🖼️ Visual illustrations and gallery
- ❓ Frequently Asked Questions
- 🖨️ Printable dental guide
- 📩 Contact form for user queries
- 📱 Fully responsive design
- ⚡ Fast performance with Vite

---

## 🛠️ Tech Stack

- React 19

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/ai-dental-health-survey-guide.git
```

### Navigate to the project

```bash
cd ai-dental-health-survey-guide
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:3000
```

---

## 📦 Build for Production

```bash
npm run build
```
